mAIcroft

Extract interesting information about redditors from their submissions, comments, and subs. Outputs data in JSON format. Brother project to Orionmelt's Sherlock, adapting and expanding its functionality and generality.

Dependencies

  • requests
  • pytz
  • TextBlob 0.9.0

Setup

  • Run pip install -r requirements.txt to install dependencies.
  • Run python -m textblob.download_corpora to download TextBlob corpora.

Usage

python maicroft.py <reddit-username>

Example

Command:

python maicroft.py thundergolfer

Output:

**Processing u/thundergolfer and returns a json string of all extracted profile attributes**
